En estos archivos se encuentra alojado el package contenedor del juego en pharo, para poder correr el juego se debe contar con la versión 7.0 de Pharo.
Para poder implementar el código dentro de la VM hay que arrastrar el archivo ".st" dentro de la ventana de la VM y elegir la opción filein.
Para ejecutar el juego original se debe abrir el "playground" y se debe escribir "JuegoDeLaVidaMorph new openInWorld" y ejecutar.
Para ejecutar el juego modificado con la regla extra se debe escribir "JuegoDeLaVidaExtraMorph new openInWorld" y ejecutar.
